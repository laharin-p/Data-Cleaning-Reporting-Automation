# 📊 Data Cleaning & Reporting Automation using Python & Power BI

## 🚀 Project Overview

In today's data-driven business environment, raw data often contains missing values, duplicate records, inconsistent formatting, and other quality issues that can lead to inaccurate analysis and poor decision-making.

This project demonstrates an **end-to-end automated data cleaning and reporting pipeline** using **Python (Google Colab)** and **Power BI**. The workflow transforms raw sales data into a clean, analysis-ready dataset and presents key business insights through an interactive Power BI dashboard.

The project showcases practical data preprocessing techniques, automation, and business intelligence skills commonly used by data analysts.

---

## 🎯 Project Objectives

* Automate the data cleaning process using Python
* Detect and remove duplicate records
* Handle missing and inconsistent data
* Standardize and transform data for analysis
* Generate automated business summaries
* Build an interactive Power BI dashboard
* Improve reporting accuracy and efficiency

## 🧹 Data Cleaning Process

The automation pipeline performs the following tasks:

* Imported raw sales data
* Explored dataset structure and quality
* Identified and removed duplicate records
* Handled missing values
* Standardized text formatting
* Converted date columns to proper datetime format
* Created additional features:

  * Year
  * Month
  * Quarter
  * Shipping Days
* Validated data consistency
* Exported a clean, analysis-ready dataset
